## 🎨 设计系统

### 颜色主题
```css
Primary（主色）:
  - 50:  #f0f9ff  (极浅蓝)
  - 500: #0ea5e9  (标准蓝)
  - 600: #0284c7  (深蓝)
  - 700: #0369a1  (更深蓝)

Secondary（辅助色）:
  - 灰度系列 (slate)
```

### 自定义组件类
```css
.btn-primary     - 渐变按钮
.card            - 卡片容器
.input-modern    - 现代输入框
.modern-table    - 现代表格
.modern-tree     - 现代树形菜单
```

### 动画效果
- ✨ `fade-in` - 淡入动画
- 📊 `slide-in` - 滑入动画  
- 🎈 `bounce-subtle` - 轻微弹跳
- 🌊 `blob` - 气泡浮动（登录页）
